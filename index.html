About TechVaaganza

What is TechVaganza?

TechVaganza is an intensive 3-day technical event focused on Amazon Web Services (AWS) and cloud computing technologies. Designed for students passionate about learning cutting-edge cloud technologies and DevOps practices.

Our Mission

TechVaganza is conducted every year with a singular focus: to provide students with hands-on experience in AWS cloud computing, industry-standard DevOps practices, and real-world cloud architecture. This is a learning-first event where students can expand their technical knowledge and stay ahead in the ever-evolving tech landscape.

Event Highlights

What makes TechVaaganza unique

AWS Workshops

Hands-on workshops covering core AWS services including EC2, S3, Lambda, RDS, and more. Learn by building real projects.

Cloud & DevOps

Deep dive into cloud architecture, CI/CD pipelines, infrastructure as code, containerization, and DevOps best practices.

Hands-on Learning

Practical, project-based learning approach. Build and deploy applications on AWS infrastructure during the event.

Expert Talks

Industry professionals and AWS certified experts share insights, career guidance, and real-world use cases.

Learning Focused

This is a learning-first event. No prizes or certificatesjust pure knowledge and skill development in AWS technologies.

3-Day Event Timeline

Your journey through AWS and cloud computing

Day 1

Cloud & AWS Foundations + Portfolio Mindset

Building strong cloud fundamentals and a portfolio mindset

  • What is cloud computing and why it matters today
  • Traditional IT vs Cloud (cost, scalability, speed)
  • Cloud service models: IaaS, PaaS, SaaS
  • Public cloud overview & shared responsibility model
  • AWS global infrastructure: regions and availability zones
  • Core AWS services overview and AWS Free Tier
  • Building a cloud portfolio and testing locally
  • Hosting a static portfolio website on AWS S3
Day 2

AI Foundations & Modern Cloud Applications

Understanding AI concepts and building AI-powered solutions on AWS

  • What Artificial Intelligence is and how it works
  • Difference between ML, Generative AI, and traditional applications
  • Where AI fits in cloud platforms
  • Real-world AI use cases students already use
  • Core AI terminology: LLMs, RAG, Foundation Models
  • Introduction to Kiro and its role in the AI lifecycle
  • How tools like Kiro accelerate AI application development
Day 3

AI-Powered Applications on AWS

Building and deploying AI-driven applications using AWS services

  • Introduction to AWS Bedrock and foundation models
  • AWS Lambda for serverless backend logic
  • API Gateway for exposing AI services
  • Integrating Bedrock with Lambda and APIs
  • Hosting and accessing AI applications over the internet
  • End-to-end AI-powered chatbot architecture
  • Final wrap-up and learning outcomes
